Get Started

  1. Install Node.js (+ NPM)
  2. Buat akun di NPM
  3. Login NPM di terminal 
    npm adduser

Publish Package

  1. Buat Github repo, init local repo 
    git init
  2. Init project 
    npm init
  3. Mulai coding, kemudian add -> commit -> push
  4. Publish package 
    npm publish

CommonJS vs ES Modules 


                /* CommonJS import */
                const express = require('express')
                
                /* CommonJS export */
                module.exports = {}
              

                /* ES Modules import */
                import express from 'express'
                
                /* ES Modules export */
                export default {}

Update Package

  1. Update our code, add -> commit
  2. Update version 
    npm version major / minor / patch
  3. Push our code
  4. Publish again 
    npm publish

Add Beta

  1. Update our code, add -> commit
  2. Change version in package.json 
    "version": "1.2.4-beta.0"
  3. Push our code
  4. Publish labeled package 
    npm publish --tag beta

Semantic Versioning

1     .      6     .      3

major                         minor                         patch

breaking change      add new feature      bug fix / perf.
