Get Started

  1. Install Node.js (+ NPM)
  2. Create account at NPM
  3. Login NPM in terminal 
    npm adduser
  4. Recheck (optional)
    npm whoami

Publish Package

  1. Create Github repo, init local repo 
    git init
  2. Init project -> code -> add -> commit -> push 
    npm init
  3. Add tag 
    
                  git tag 1.0.0
                  git push --tags
  4. Publish package
    npm publish

CommonJS vs ES Modules 


                /* CommonJS import */
                const express = require('express')
                
                /* CommonJS export */
                module.exports = {}
              

                /* ES Modules import */
                import express from 'express'
                
                /* ES Modules export */
                export default {}

Update Package

  1. Update our code, add -> commit
  2. Update version 
    npm version major / minor / patch
  3. Push tags and our code
  4. Publish again 
    npm publish

Add Beta

  1. Update our code, add -> commit
  2. Change version in package.json 
    "version": "1.2.4-beta.0"
  3. Add -> commit, add tag 
    git tag 1.1.0-beta.0
  4. Push tags and our code
  5. Publish labeled package 
    npm publish --tag beta
