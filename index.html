Prequisites

  • Familiar dengan NPM

Get Started

  1. Install Node.js (+ NPM)
  2. Buat akun di NPM
  3. Login NPM di terminal 
    npm adduser

Publish Package

  1. Buat Github repo, init local repo 
    git init
  2. Init project 
    npm init
  3. Tambahkan Github remote
  4. Mulai coding, kemudian add -> commit -> push
  5. Publish package 
    npm publish

Update Package

  1. Update our code, add -> commit
  2. Update version 
    npm version major / minor / patch
  3. Push our code
  4. Publish again 
    npm publish

Semantic Versioning

1     .      6     .      3

major                         minor                         patch

breaking change      add new feature      bug fix / perf.
